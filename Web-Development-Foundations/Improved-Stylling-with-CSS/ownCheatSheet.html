My Own Cheat Sheet

Some CSS Font Properties

Property Name Description
font-family Font Family Especifica uma lista priorizada de um ou mais nomes de famílias de fontes e/ou nomes de famílias genéricas para o elemento selecionado
font-size Font Size Define o tamanho da fonte. Alterar o tamanho da fonte também atualiza os tamanhos das unidades <lenght> relativas ao tamanho da fonte, como em, ex e assim por diante
font-stretch Font Stretch Seleciona uma aparência normal, condensada ou expandida de uma fonte
font-style Font Style Define se uma fonte deve ser estilizada com o estilo normal, itálico ou oblíquo de sua família.
font-weight Font Weight Define a espessura (ou negrito) da fonte. As espessuras disponíveis dependem da família de fontes que está definida atualmente.
font-optical-sizing Font Optical Sizing Define se a renderização de texto é otimizada para exibição em tamanhos diferentes.
